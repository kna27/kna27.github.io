Repetition Timer

2021

A timer that repeats over an interval.

C#, Unity

GitHub Repo

APK Download


This is a timer app I made in Unity mainly over a weekend. It is useful for presentations with a set length per slide. It helps you keep track of how much time you have left in the repetition and includes a progress bar.

I am considering putting it on the Android Play Store in the future.
